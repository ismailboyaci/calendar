# Kodun Çalıştırılması

- Dosyayı indirin.
- Dosyanın bulunduğu klasörde windows terminali açın.
- Terminale `docker-compose up` kodunu yazın.
- Daha sonra tarayıcıda (http://localhost:3000/) adresini girin.
